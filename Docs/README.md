## Getting Started

### New to the package?

1. Read the [**FAQ**](FAQ.md) for common questions
2. Check the [**Usage Guide**](Usage.md) for API examples
3. Open the Language Editor (`Tools > Localization > Language Editor`)
4. See [**Keybinds**](Keybinds.md) for keyboard shortcuts

### Want to understand the internals?

- Read [**How It Works**](HowItWorks.md) for architecture overview
- Check [**BLOC Format**](BLOC_FORMAT.md) for binary format details

### Having issues?

- Check the [**FAQ Troubleshooting section**](FAQ.md#troubleshooting)
- Review [**How It Works - Performance Tips**](HowItWorks.md#performance-tips)
